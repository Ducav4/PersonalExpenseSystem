# Personal Expense System

Progetto finale di Programmazione: Sistema di Gestione delle Spese Personali.
Sviluppato in Python con database SQLite.

## Requisiti per l'esecuzione
* **Interprete**: Python 3.x
* **Librerie**: `sqlite3` (inclusa nella standard library di Python, nessuna installazione richiesta).

## Istruzioni per eseguire il programma
1. Aprire il terminale (o prompt dei comandi).
2. Spostarsi nella cartella del progetto:
   `cd PersonalBudgetSystem`
3. Eseguire il comando di avvio:
   `python src/main.py`
