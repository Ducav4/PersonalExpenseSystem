# Personal Expense System

Questo software è un'applicazione da console per la gestione delle finanze personali.


## Requisiti di Sistema
Per eseguire il software è necessario avere installato:
* **Interprete:** Python 3.x (versione 3.6 o superiore consigliata).
* **Librerie:** Il progetto utilizza solo la libreria standard `sqlite3`, già inclusa in Python. Non è necessaria alcuna installazione aggiuntiva.

## Istruzioni per l'Esecuzione
Seguire questi passaggi per avviare il programma:

1.  Scaricare e decomprimere la cartella del progetto.
2.  Aprire il terminale (o Prompt dei Comandi).
3.  **Navigare all'interno della cartella del progetto.**
    * *Suggerimento:* Digitare `cd ` (seguito da uno spazio) e trascinare la cartella `PersonalExpenseSystem` dentro la finestra del terminale, poi premere Invio.
4.  Eseguire il comando di avvio:
    
    python src/main.py
   

## Struttura del Repository
* **src/**: Contiene il codice sorgente Python (`main.py`).
* **sql/**: Contiene lo script SQL per la creazione del database e i dati di esempio.
* **demo/**: Contiene il video dimostrativo del funzionamento.
* **README.md**: Questo file di documentazione.
